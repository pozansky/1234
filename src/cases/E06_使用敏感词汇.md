# E06 使用敏感词汇

## Definition

仅当出现本条规定的敏感词根、固定短语或模板短语时判违规；未命中本列表一律不按本条违规处理。

## Bad cases

- 这次就是翻身机会，跟上就行。
- 现在就是捡钱行情，赶紧进场买票。
- 这票成功率90%，收益率30%。
- 这一波抓10倍股，单车变摩托。
- 改变账户，从此稳赚不赔。

## Good cases

- 仅做中性行情描述，未出现本条敏感词列表。
- 仅提示风险控制（如“控制仓位、注意止损”）。
- 仅介绍课程/服务流程，未出现本条敏感词列表。

## 易错说明

- **易误判**：本条为“白名单外不判违规”规则。只有命中本文件列出的敏感词根/固定短语/模板短语才加分，其他表达一律不按 E06 违规处理。
- **易漏判**：客服常用空格、标点、表情拆词规避（如“翻 身”“回 本”“稳 赚 不 赔”）。拆开写法仍按命中处理。
- **固定短语口径**：
  - 固定短语需完整连续：`预期空间`、`进场买票`、`跟上买票`、`上车买票`、`安排做票`、`吃肉行情`、`捡钱行情`、`牛股`、`妖股`、`改变账户`、`稳赚不赔`、`本金无忧`、`持续盈利`、`钱生钱`、`赚回服务费`、`抓10倍股`、`单车变摩托`、`翻身机会`、`游资机构抱团股`、`抱团股`。
  - 模板短语：`成功率*%`、`收益率*%`、`连续*个涨停`、`*字板`。
- **打分口径（固定分值）**：
  - 高风险：`稳赚不赔`、`本金无忧`、`持续盈利`、`钱生钱`、`抓10倍股`、`单车变摩托`，固定 +60。
  - 中风险：`翻倍`、`翻番`、`暴涨`、`捡钱行情`、`吃肉行情`、`牛股`、`妖股`，以及模板 `成功率*%`、`收益率*%`、`连续*个涨停`、`*字板`，固定 +35。
  - 低风险：`预期收益`、`翻身`、`回血`、`回本`、`预期空间`、`翻身机会`、`进场买票`、`跟上买票`、`上车买票`、`安排做票`、`赚回服务费`、`游资机构抱团股`、`抱团股`、`改变账户`，固定 +15。

## Notes

- 本条只管“是否命中敏感词列表”，不做扩展联想。
