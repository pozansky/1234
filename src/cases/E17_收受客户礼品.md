# E17 收受客户礼品

## Definition

接受或暗示接受客户赠送的礼品、红包、请客等，可能影响专业判断或违反公司/监管关于受赠的要求。

## Bad cases

- 你给我寄点特产过来就行了
- 下次我去你城市，你请我吃个饭
- 给我发个红包意思一下

## Good cases

- 明确拒绝客户赠礼，说明公司有相关规定
- 建议客户将心意转化为对服务改进的建议反馈
- 上面的抽奖可以参与一下，免费的，抽中什么了记得和我说
- 第一种方案是让别人给你微信上再发一百九十八块钱红包
- 还主动给咱们助教发红包，让她也沾沾喜气[呲牙]但是咱们正规公司，不能收取额外的钱的
- 姐，有任何福利我都会第一时间给到您的

## Risk keywords

- 礼物
- 红包
- 赠送
- 收受
- 寄给您
- 一点心意
- 寄个东西
- 给您带点
- 寄点特产
- 一点小意思
- 请我吃饭
- 到你的城市玩
- 你请客

## Strong protection keywords

- 拒绝赠礼
- 公司规定

## 易错说明

- **总结口径**：本条核心看“员工是否索要/接受客户红包礼品”，不是“只要出现红包二字就违规”。
- **易误判**：客户资金不足时，员工提示可周转资金，未向客户本人索要红包，不按本条违规处理。
- **易误判**：客户主动发感谢红包，员工明确提示“正规公司不能收取额外的钱”并拒收，不按本条违规处理。
- **易误判**：提示客户参与免费抽奖、活动福利，并让客户反馈抽奖结果，属于活动互动或服务跟进；只要未要求客户将奖品、红包、礼物转给员工本人，不按本条违规处理。
- **易误判**：向客户提供福利，未存在接受客户礼品的行为，合规
- **易漏判**：若员工主动索要红包、暗示收礼、或接受客户红包礼品，应按本条违规处理。

## Notes

- 只要出现“收礼”“请客”并与客服个人挂钩，一般都应视为高风险。
