# E13 违规指导

## Definition

无证券投资顾问执业资质的人员，在 **当前对话中** 向客户下达 **可直接执行的个股交易指令**。

【重要过滤规则 - 优先级最高】
命中以下任意关键词，必须先判定为“服务介绍”，禁止判违规：
- 什么价格区间买进
- 什么价格区间卖出
- 会给你 / 会给到 / 会告诉你
- 会推送 / 会提示 / 会通知
- 合作后 / 加入后 / 内部
- 系统会 / 软件会 / 老师会
- 盘中会 / 都会给到 / 都会发
- 统一推送 / 内部会员 / 服务期 / 体验期
- 带你 / 跟上 / 价格区间 /指导提示 / 

E13 必须同时满足以下 **三个条件**：

1. **具体标的**：明确指向某只股票（“这只”“这个票”“代码XXX”“某股票名称”）
2. **交易动作**
   - 明确的交易行为
   - 示例：
     - 买入
     - 卖出
     - 加仓
     - 减仓
     - 持有
     - 清仓
     - 止损
     - 止盈
     - 做T
3. **可执行条件**
   - 指令包含明确执行条件，使客户可以 **直接按照指令进行交易**
   - 示例：
     - 时间（现在 / 明天 / 开盘 / 收盘）
     - 价格（8元买入 / 跌破卖出）
     - 仓位（全仓 / 半仓 / 30%仓位）

只有 **同时满足以上三项**，才构成违规指导。

---

## 核心原则

**E13 只识别“当前对话中已经下达的、客户可直接执行的个股交易指令”。**

### 先排除，再判断

凡是以下情况，**直接排除 E13，不再进入三要素判断**：

- 服务介绍
- 软件功能说明
- 合作流程说明
- 合作后会提供的消息/推送/提醒说明
- 案例展示
- 投资观点
- 风险提示
- 转述他人观点
- 操作方法介绍（未指定个股）
- **复盘描述（过去发生的操作）**
- **收益回顾**
- **假设性表达**

---

## 服务介绍类（重点排除）

即使句中出现“买入、卖出、加仓、减仓、止盈止损、价格区间、代码”等高风险词，只要核心语义是介绍服务、软件、合作后会提供的内容，就**直接判不属于 E13**：

- 会告诉你买哪一支股票，什么价位买入，什么价位卖出
- 会给到你什么价格区间买、什么价格区间卖
- 合作后会推送股票代码、买卖区间、止盈止损位
- 软件会提示买卖点
- 老师会通知你什么时候加仓减仓
- 建仓点位、止盈止损区间都会给到你

**原因：**  
这些表述是在描述“未来会提供什么服务/信息”，不是在 **当前对话中** 直接对某只股票下达交易指令。  
客户此时仍需等待后续推送、老师通知或软件提醒，**不能仅根据当前这句话立即执行交易**。

---

## 当前可执行判定标准

只有当员工在 **当前这句或当前轮对话中**：

- 已经明确某只股票  
- 已经给出交易动作  
- 已经给出执行条件  

且客户 **无需等待后续推送/老师通知/软件提醒**，即可直接操作，

才可能构成 E13。

否则一律不构成。

---

## 判断流程（强制执行）

### Step 1：语境分类（四选一）

A. 当前交易指令  
B. 服务介绍 / 软件功能说明 / 合作后推送说明  
C. 历史复盘 / 案例展示 / 收益回顾  
D. 一般观点 / 风险提示 / 其他  

**重要说明**：
- "当前交易指令"包括：明确指令（"买入"、"卖出"）**和建议语气**（"可以持有"、"可以考虑买入"、"建议关注"）
- 只要包含具体标的、交易动作、可执行条件，即使是建议语气，也属于"当前交易指令"
- 区分"操作建议"和"行情观点"：
  - 操作建议：告诉客户"可以持有"、"可以买入"、"可以卖出" → 属于 A
  - 行情观点：只说"这个位置有支撑"、"可能会涨"，没有明确的操作建议 → 属于 D

---

### Step 2：语境过滤

- 若为 **B / C / D → 直接判定：不构成 E13**
- 只有 **A → 继续判断三要素**

**注意**：不要因为"可以"、"建议"等词语就认为是"行情观点"。关键看是否包含三要素。

---

### Step 3：三要素校验

必须同时满足：

- 具体标的 ✅
- 交易动作 ✅
- 可执行条件 ✅

三项全部满足 → **构成 E13**  
任一缺失 → **不构成 E13**

---

## Bad cases

- 东软集团明天开盘买入。
- 贵州茅台现在可以加仓。
- 600036 跌破 8 元卖出。
- 东软集团现在不要卖，等我通知。
- 000001 明天半仓买入。

---

## Good cases

- 合作后系统会推送短线案例、股票代码、买卖区间和止盈止损位到您手机上。
- 我们这边是直接告诉你买哪一支股票，什么价位买入，什么价位卖出，这种服务给到你。
- 给到你单独的个股代码、什么价格区间买、什么价格区间卖。
- 合作以后每个交易日盘中都会实时推送，明确的代码跟买卖区间指导都有，您收到以后直接挂单操作就可以。
- 买、卖直接提示，怎么简单怎么来，买卖区间价位都有提示，简单跟着照做就行。
- 建仓点位都会给到你，有人专门负责给你盯盘，通知你加仓减仓。
- 内部就是这样的，直接单独明确的个股代码、什么价格区间买、什么价格区间卖，直接发您。
- 我们内部的话，每天都有这个短线的黄金变模型案例给到你的，代码买卖线全都有了，止盈止损位都是发在你自己的手机上面的。
- 止盈止损区间都会给到你。
- 拿不准的话，你明天先观察一下走势，不要一开盘就立马去买入。（风险提示，无标的）
- 姐姐，尾盘2:50左右买入嘛，这样浮动比较小，明天看机会随时可以走。（操作方法介绍，无标的）
- 因为就是我们这边如果说被套得深的话，我们这边合作之后是有辅助咱们来做 T 的指标的，就是咱们可以做 T 压缩成本嘛，对吧？（介绍服务功能）
- 我帮您把问题提交给老师，具体买卖建议由持证老师回复您。
- 这是老师原文观点，我不做修改，您同时结合风险提示参考。
- 我只负责转达老师原话，具体操作您自己决定。
- 老师说的这个票，您参考一下老师观点。（无“现在就买”类指令）
- 您现在2000的仓位有吗？（询问资金情况）
- 您现在调整1000~2000出来，可以直接跟上。（“跟上”指服务，非股票交易）
- 您300开头的票可以买吗？（询问权限）
- 您喜欢做短线还是中线？（询问偏好）
- 回复【6】了解合作详情。（引导回复）
- 只做一般性风险提示或投资理念分享。
- 引导客户根据公开研究报告和自身情况决策。
- 高层释放宽松信号，市场向好预期很强。（行情观点）
- 科技方向中长期向好预期不变。（行业观点）

---

## 特别规则

- 过去时（如：周五买、今天卖）→ 不算违规  
- “会提供买卖点/会推送代码” → 不算违规  
- 服务能力说明 ≠ 当前交易指令  
- 客户需要等待后续信息 → 不算违规  

---

## 兜底原则（非常关键）

**只要存在合理解释表明该句是在介绍服务，而不是当前直接指导交易 → 一律不判 E13**

 **宁可放过，也不要误判服务话术**
